# Pilot Invitation Email

Subject: Quick classroom module on AI, careers, and human value

Hi [Name],

I am piloting a classroom module based on my book, *After the Grind*, and I would value your feedback.

The module helps students answer a question I keep hearing in different forms:

If AI can do much of the routine work students expected to do early in their careers, what kind of human value should they build now?

The pilot includes:

- A free 15-question Human Work Spectrum assessment.
- A personal archetype result for each student.
- A 45-minute lecture outline.
- A 75-minute class plan.
- A group activity.
- A short student assignment called "My Post-Grind Value Plan."

The materials are here:

https://humanworkspectrum.com/classroom

If you are willing to run it in a class or review it as a possible module, I would be grateful for direct feedback:

1. Is the concept clear to students?
2. Is the class plan easy to run?
3. What would make it more useful for faculty?
4. Would class-level aggregate results be valuable?
